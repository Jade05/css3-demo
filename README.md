#具体查看方式
目前暂时支持直接输入url，回车查看。
在本地安装好后，启动服务，再在浏览器中输入本机的地址即可访问

如果的如果，您没有安装好node还是依旧想查看demo，将项目中的views/parts/下的ejs文件单独拿出来，将'.ejs'后缀改为'.html'，然后记得在该html文件中引入jquery的包，直接就可以访问。
